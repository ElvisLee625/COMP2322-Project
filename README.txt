How to Set Up the Web Server

1. Make sure all the files below are in the same folder
- Project.py
- test.txt
- image.jpg
- index.html

2. Open the terminal and type the full path to your Project.py
- example: C:\User\xxx\xxx\xxx\xxx\Project.py

3. You are now set

4. To stop the server: Press Ctrl + C in the terminal.

-----------------------------------------------------------------

How to access the webpage
 
1. Use one of the URLs:
- http://127.0.0.1:8080/             (Home Page)
- http://127.0.0.1:8080/test.txt     (Text File)
- http://127.0.0.1:8080/image.jpg    (Image File)

------------------------------------------------------------------

log file (webServer.log)
- In the same folder of Project.py
- The file will be auto generated  
- Record the client request
- Format: client IP, access time, requested file name, response type
